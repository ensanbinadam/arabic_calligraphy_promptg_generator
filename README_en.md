# Arabic Calligraphy Prompt Generator for Names and Phrases

An interactive Arabic interface for generating professional prompts to design **Artistic Arabic Calligraphy** for names and short phrases. With multiple, organized options, it helps users create clear, precise, and ready-to-use prompts for AI image generation tools.

This project aims to facilitate the creation of Arabic calligraphy prompts professionally, without the need to write every detail manually each time.
The user simply types the name or phrase, then selects from available menus such as: **text type, composition shape, script type, colors, background, decorations, rendering style, and detail level**. The result is a fully integrated Arabic prompt, an English prompt, and a **Negative Prompt** to enhance output quality.

---

## Project Concept

This generator is dedicated to creating prompts for Arabic typographic designs, whether it is:

- A single name
- A two-word name
- A three-word name
- A short phrase
- A logo or brand name
- Artistic text with a spiritual, luxurious, or modern vibe

The project revolves around **converting visual and artistic choices into structured textual descriptions** that help AI generation tools produce compositions closer to the user's vision.

---

## Features

- Easy and organized Arabic interface.
- Free text input for any name or phrase.
- Support for single, double, triple names, and short phrases.
- Multiple options to customize the prompt's appearance.
- Generation of:
  - **Arabic Prompt**
  - **English Prompt**
  - **Negative Prompt**
- Quick copy buttons for each output.
- Ability to copy all outputs at once.
- Download results as a text file.
- Save settings locally within the browser.
- Restore saved settings or reset to default.
- Design suitable for hosting via **GitHub Pages**.
- Works directly from the browser without the need for a server or complex setups.

## Available Generator Options

### 1. Text Type

To determine the nature of the input, such as:

- Single name / Two-word name / Three-word name
- Short phrase / Short verse
- Logo or brand name

### 2. Composition Shape

To determine the overall silhouette of the design:

- Droplet / Flame / Vertical Oval / Circular Emblem
- Vertical / Horizontal / Cohesive Freeform

### 3. Script Type

To give the prompt the appropriate typographic feel:

- Free expressive inspired by Diwani / Thuluth / Hybrid
- Soft Kufic / Modern Ornamental

### 4. Interweaving Degree

To define how the letters overlap:

- Light / Medium / Strong / Full Fusion

### 5. Readability Level

Balancing aesthetics with legibility:

- More artistic than readable / Balanced / Extremely clear / Logo-friendly

### 6. Colors & Backgrounds

Various options including Luxurious Gold, Pure White, Royal Black, Transparent, Night Blue, and more.

### 7. Decorations & Rendering

Options include soft glows, Islamic motifs, crescents, embossed gold, 3D rendering, and metallic engraving.

## How to Use

1. Open the page in your browser.
2. Type the name or phrase in the designated field.
3. Choose the appropriate settings from the dropdown menus.
4. Click the **Generate** button.
5. The generator will produce the Arabic, English, and Negative Prompts.
6. You can then copy, download, or save your settings for later use.

---

## The Value of the Negative Prompt

The project also generates a **Negative Prompt** to help reduce common errors during image generation, such as:

- Distorted Arabic letters
- Poor readability or overcrowded compositions
- Random decorations or repeating elements
- Broken font structure or additional/incorrect letters
